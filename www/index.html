Middle Earth Navigator
A chart demo for Navigraph

Summary

Click and drag or use the arrow keys to navigate around the chart. Zoom using the mouse wheel or [shift] + [arrow up/down].

Clicking a location pin will display information about the place.

Thanks to Tolkien Gateway for help with locating and describing places.

Technical info

Several technical decisions were made to maximise app performance:

  1. Moving all draw operations into a web worker to free up the main UI thread.
  2. Making use of the window.requestAnimationFrame function to draw at the highest possible framerate.
  3. Cropping the map image before drawing to avoid drawing off-screen content.

Note: take a look at the repository for more details.
