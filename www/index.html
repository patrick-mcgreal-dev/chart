Middle Earth Navigator
A chart demo for Navigraph

Summary

Click and drag or use the arrow keys to navigate around the chart. Zoom using the mouse wheel or [shift] + [arrow up/down].

Clicking a location pin will display information about the place. Hold [shift] and click to place your own pins on the chart.

Thanks to Tolkien Gateway for help with locating and describing places.

Note: take a look at the code repository for more details.

App performance

Several technical decisions were made to maximise app performance:

  1. All draw operations take place in a web worker to free up the main UI thread.
  2. The window.requestAnimationFrame function is used to draw at the highest possible framerate.
  3. The map image is cropped before drawing to avoid drawing off-screen content.
